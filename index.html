jChat

This is a real time LAN Chat application developed using Node.js

View project onGitHub

Welcome Buddy!

Download the project and make sure that you have Node.js installed. Use the package.json to install the required modules by running the command: npm install

Run the index.js by giving the command node index.js

Open up your browser and type in : http://localhost:8000 and at the same time ask your friends on your LAN to point their browser to the above URL.

And Enjoy Chatting!
